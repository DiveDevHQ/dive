Prerequisite

You need to have the following data ready:

  • Client Id
  • Client Secret

To get "client id" and "client secret", please create a "connected app" under "App Manager" in Salesforce. Enable OAuth settings. For OAUTH scopes, select "Manage user data via APIs (api)" or whatever meets your needs, make sure to also select "Perform requests at any time (refresh_token, offline_access)".
If you can not find "App Manager", check your permissions here.
